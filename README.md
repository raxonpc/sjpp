# sjpp (SJP++)

C++ API for [Słownik Języka Polskiego](https://sjp.pl/)

It can be used as a backend for a dictionary, word-game, etc.

## How to build

```sh
cd sjpp
mkdir build
cd build
cmake ..
cmake --build .
```

## Usage

Just include the aggregate header file
```cpp
#include <sjpp/sjpp.hpp>
```
